hexo 에서 github 으로 deploy 할때 암호 입력 없이 진행하는법

2020-02-09 01:34:22

참고 링크: https://soulsearcher.github.io/2019/08/09/hexo-deploy-failed/

참고 링크는 이중인증에 대한 내용이지만
이중인증과 상관없이 사용가능하다.
암호 매번 입력하는것도 번거럽고

ssh key 만들기

git 깔고
git bash 를 이용했다.

이러면 .ssh 폴더와
id_rsa
id_rsa.pub
가 생긴다.

ssh config 추가

내가할땐 UseKeychain yes 부분이 문제가 되서 주석처리했다.

github 에 ssh key 등록하기

우측 상단 의 프로필 아이콘 -> Settings -> SSH And ~~ -> SSH 키 등록 id_rsa.pub 내용을 복분한다.

암호 없는 ssh 확인

id 도 쓸필요 없이 연결해본다.

github 는 ssh 명령을 던지지 못하게 막혀있지만
접속자가 누구인지 인지하면 성공

hexo 의 _config.yml 수정

기존의 https://github~~ 주소가 아니고
git@ 로 시작한다.
마지막은 .git 으로 끝난다.

hexo 로 배포해보자

암호없이 잘 넘어가면 성공

hexo
node.js 에서 tcp 서버 샘플
